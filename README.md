# Galbraith Forge

Single-page marketing site for Galbraith Forge LLC — custom software and technology investing.

Static HTML with a [Vanta.js](https://www.vantajs.com/) `CLOUDS2` animated background. No build step.

## Develop

Open `index.html` directly, or serve the folder:

```sh
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deploy

Hosted on [Vercel](https://vercel.com/) as a static site — pushes to `main` deploy automatically.
